# 🐱 Cat Quote App

A simple web app that displays random cat images with custom messages.

## What It Does

1. You enter a **tag** (like "cute", "orange", or "fluffy")
2. You enter some **text** (like "Hello World!")
3. Click the button
4. A cat appears with your message on it!

## How to Use

- Open the app in your browser
- Type in a cat tag
- Type in what you want the cat to say
- Click "Get Cat" button
- Enjoy your custom cat image!

## Popular Tags

Try these: `cute`, `orange`, `fluffy`, `grumpy`, `lazy`, `white`, `black`

## Setup

Just clone the repo and open `index.html` in your browser. That's it!

```bash
git clone https://github.com/Ruturaj2007/wapproject.git
cd wapproject
```

Then open `index.html` in your browser.

## API

This app uses the free **[CataaS API](https://cataas.com/)** to get cat images with your custom text.
